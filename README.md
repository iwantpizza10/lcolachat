# Lcolachat!
A locally hosted chat app built with Slint & Rust because of course it is.

## How to Use
Download the binary from the Releases section, and run it.
From there, you should see a screen with two buttons: "Connect" and "Host."
If you wish to connect to a chatroom another user (or app instance) is hosting, select Connect.
If you would like to create a chatroom for others to connect to, select Host.

From there, you can type in your username and either the IP to connect to or the chatroom name that others will see.
If you do not know an IP address, you can find it in your OS's settings app or just use 'localhost' if you're connecting to your own computer.

Please note that the **server** must be **left open & running** in order to be connected to.
If you need to connect to your own computer, simply open the app *twice*.
